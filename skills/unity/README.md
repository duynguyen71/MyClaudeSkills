# Unity Skills

Skills for Unity game development workflows.

## Structure

This namespace contains the Claude Code Game Studios framework - a complete game development system with agents, skills, hooks, and documentation.

### Framework Location

`Claude-Code-Game-Studios/` - Complete game development framework

### Framework Components

- **Agents** (`.claude/agents/`) - 48 specialized role-based agents for game development
- **Skills** (`.claude/skills/`) - 37 workflow-based skills for common tasks
- **Hooks** (`.claude/hooks/`) - Automation scripts for validation and session management
- **Rules** (`.claude/rules/`) - Code quality and standards enforcement
- **Docs** (`.claude/docs/`) - Architecture and coordination documentation

### Available Skills

See `Claude-Code-Game-Studios/.claude/skills/` for the complete list of workflow skills including:
- brainstorm, design-system, prototype
- code-review, design-review, balance-check
- sprint-plan, milestone-review, gate-check
- team-combat, team-ui, team-narrative, team-audio
- And many more...

### Available Agents

See `Claude-Code-Game-Studios/.claude/agents/` for the complete list of specialist agents including:
- Unity specialists (unity-specialist, unity-dots-specialist, unity-shader-specialist, etc.)
- Programmers (gameplay, AI, network, tools, engine)
- Designers (game, systems, level, narrative, economy)
- Artists (art director, technical artist, audio director)
- Production (producer, QA lead, release manager, DevOps)

## Usage

To use this framework in a Unity project, copy the `Claude-Code-Game-Studios/` directory to your project root or reference it from your project's CLAUDE.md.
